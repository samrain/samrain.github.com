Emacs 中将文件编码转换为 UTF-8

2016-05-18
gif , 视频转换

打开一个文件后键盘依次输入

C-x C-m f utf-8 RET

或者用鼠标右键点击最左下角那个字母，比如 C ，这时会显示 Coding system for saving file (default chinese-gbk-dos):
然后输入 utf-8 后回车即可。
