配置ubuntu上拼音输入法Fcitx5

因为搜狗输入法只能支持到Ubuntu2004版，百度输入法也同样不可用。为了能方便输入，所以在Ubuntu2204上安装Fcitx5。

  1. 把fcitx4删除干净 sudo apt remove fcitx*
  2. 安装fcitx5 sudo apt install fcitx5
  3. 下载肥猫wiki词库文件zhwiki.dict，下载地址： https://github.com/felixonmars/fcitx5-pinyin-zhwiki/releases
  4. 词典文件复制到~/.local/share/_fcitx5_/pinyin/dictionaries。
  5. 重启fcitx5
