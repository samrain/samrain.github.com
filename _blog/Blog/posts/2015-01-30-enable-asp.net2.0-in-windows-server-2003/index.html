在 Windows Server 2003 中启用 ASP.NET 2.0

Posted on Jan 30, 2015

##在 IIS 中的"Web 服务扩展"选项卡上启用

  1. 单击开始，指向管理工具，然后单击Internet Information Services (IIS) 管理器。
  2. 在导航窗格中，在服务器名下单击Web 服务扩展。
  3. 找到ASP.NET v2.0.50727，请单击，然后单击允许。

##CLI命令启用

  • Asp.NET
  • SPS
  • win2003
