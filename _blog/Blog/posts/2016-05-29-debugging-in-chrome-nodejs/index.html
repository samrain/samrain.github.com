在Chrome中调试nodejs

2016-05-29
Chrome , nodejs

install

安装node-inspector插件

sudo npm install -g node-inspector

Getting start

node-debug XXX.js

会自动启动chrome（Chromium）浏览器访问http://127.0.0.1:8080/debug?port=5858，这时就可以设置断点进行调试了。
