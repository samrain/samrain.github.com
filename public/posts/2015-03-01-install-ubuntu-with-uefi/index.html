怎么在UEFI电脑上安装Ubuntu

官网介绍在此

简单步骤如下：

  1. 正常通过CD或者USB安装
  2. 如果无法通过硬盘启动，那么用CD/USB引导启动，选Try Ubuntu Without Installation
  3. 安装Boot-Repair 工具，然后执行
  4. 再次硬盘启动。
