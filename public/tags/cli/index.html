转换图形文件在linux命令行方式

安装ImageMagick 将当前目录下的图片文件按照文件名排序后合成pdf convert *.png out.pdf 缩小文件尺寸 convert -resize 50%x50% *.jpg

在Ubuntu上使用命令行CLI方式使用VPN

思维导图文件下载 安装客户端 sudo apt-get install pptpd network-manager-pptp network-manager-vpnc network-manager-gnome 设置客户端 sudo pptpsetup –create vpnname –server ip –username test –password test –encrypt –start 拨号 sudo pon vpnname 使用 sudo addroute 挂断 sudo poff vpnname
